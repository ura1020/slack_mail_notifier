# slack_mail_notifier

    Slack の未読メッセージをメール通知してくれるSlack App
    メールサーバはAmazon SESの想定
    CloudWatch Eventで定期実行の想定

# 設定手順
    - Slack Appに登録
    - AWS Lambdaにデプロイ
    - AWS CloudWatchで定期実行設定
